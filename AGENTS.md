# Agents

This file documents the various agents (human or AI) interacting with the MyDentist system.

## 1. Receptionist Agent
**Type:** Human / AI Assistant
**Responsibilities:**
- Managing patient appointments (scheduling, rescheduling, cancelling).
- Patient check-in and check-out.
- Handling incoming calls and inquiries.
- Verifying insurance details.

## 2. Dentist Agent
**Type:** Human
**Responsibilities:**
- Performing dental procedures.
- Diagnosing oral health issues.
- Creating treatment plans.
- Prescribing medications.

## 3. Billing Coordinator
**Type:** Human / Automated System
**Responsibilities:**
- Generating invoices.
- Processing insurance claims.
- Managing accounts receivable.
- Sending payment reminders.

## 4. Patient Notification Agent
**Type:** Automated System
**Responsibilities:**
- Sending appointment reminders (SMS/Email).
- Sending recall notices for check-ups.
- Sending birthday greetings.

## 5. System Administrator
**Type:** Human
**Responsibilities:**
- Managing user accounts and permissions.
- System maintenance and backups.
- Troubleshooting technical issues.
